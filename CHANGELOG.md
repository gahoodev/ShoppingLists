# Changelog

## [v2.8.5.00 Released] - 2026-05-08

### Added
- XLSXおよびPDF形式のエクスポート機能を追加しました

### Removed
- 「復元」ボタンを削除しました
- ボタンやタブのデザインをシンプルにするため、グラデーションとグロー効果を削除しました

### Changed
- すべてのボタンアイコンをLucideスタイルのSVGアイコンに置き換えました

### Fixed
- 共有方式をBase64からID-based sharing systemに変更し、URLを大幅に短縮しました
- 共有URLを開いた際に、既存のローカルデータが上書きされてしまう問題を修正しました

## [2.5.0.16] - 2025-11-19
