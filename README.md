# ShoppingLists
あらゆるサイトの「欲しい」を、一つの場所に。

**ShoppingLists** は、複数のオンラインストアやECサイトに散らばっている商品を、ワンクリックで一つのリストに集約できる、シンプルで強力なブックマークツールです。
乱雑になりがちなブラウザのブックマークや、無数に開かれたタブからあなたを解放し、スマートで快適なオンラインショッピング体験を提供します。

## 概要 (Overview)
オンラインショッピングが日常となった今、私たちは様々なウェブサイトで商品を探します。しかし、情報が分散することで、以下のような問題に直面しがちです。

「あの商品はどのサイトで見たんだっけ？」と見失ってしまう。

- 複数の商品を比較するために、たくさんのタブを開きっぱなしにしている。

- ブックマークだらけになる。

- イベント（誕生日、クリスマスなど）のためのプレゼントリスト作成が面倒。

ShoppingLists は、これらの課題を解決するために生まれました。このWebツールを使って、気になった商品をあなたの統合リストへ即座に追加。商品の比較検討、購入計画、そして友人や家族との共有まで、オンラインショッピングのあらゆるプロセスを効率化します。

## 主な機能 (Key Features)

- サイトを横断した一元管理:
  - A社の公式サイトで見つけた限定品も、B社の総合通販サイトで見つけた日用品も、すべてを一つのダッシュボードで管理・閲覧できます。

- 自由なリスト作成と整理:
  -「引越しで買う家具リスト」「友人への誕生日プレゼント候補」「自分のご褒美リスト」など、目的に応じて自由にリストを作成し、商品を分類できます。

- リストの共有機能:
  - 作成したリストは、専用のURLを発行して友人や家族と簡単に共有できます。プレゼントの希望を伝えたり、共同購入の計画を立てる際に便利です.
